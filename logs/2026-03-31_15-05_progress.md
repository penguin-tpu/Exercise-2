# Progress Log

- Added a `--list-configs` CLI mode so users can discover preset names without reading the source tree.

- `perf_modeling/config/__init__.py` now exports `describe_named_config` alongside the preset registry helpers.
- `scripts/run_sim.py` now supports `--list-configs`, prints all preset names plus descriptions, and exits without running a program.
- `tests/test_run_sim_cli.py` now validates:
  - preset descriptions
  - the `--list-configs` CLI path
  - the absence of normal simulation output when listing configs
- `README.md` now includes `--config` and `--list-configs` in the common-usage section.
