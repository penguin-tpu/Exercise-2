## Current Implementation Status

- The ISA layer now also recognizes `vmax` as vector-unit work and validates it through the same timing/event path as `vadd` and `vmul`.
- The ISA layer now also recognizes `vreduce_max` as vector-unit work and validates it through the same timing/event path as `vreduce_sum`.
- Perfetto export now maps the simulator’s current opcode names onto the intended vector, MXU, and DMA tracks.

- Workload builders now also cover a float32 dense-ReLU path that chains MXU matmul, vector bias add, and vector ReLU in both direct and staged execution.
